# 项目描述

  1. 开发工具：webstrom（vscode）+plsql+sourcetree

  2. 技术栈：vue+git+webpack+twaver+element-ui+gitbook

  3. 项目描述：因为旧架构流程存在不兼容主流浏览器，页面响应速度慢的问题，所以使用全新的前端技术对其进行改造，主要包括流程引擎的组件库封装，表单组件库的封装以及流程图组件的封装。

主要负责：

（1）引擎组件库：设计流程引擎组件，工具栏组件，表单流转组件，为流程表单开发人员提供引擎数据、表单数据、全局数据和事件，并支持国际化；

（2）表单组件库：基于element-ui组件进行二次开发，注入了流程引擎数据，根据引擎数据对表单上的一些组件添加权限控制，如输入框的禁用等；同时新增一些组件，如下拉树组件，富文本组件等并支持国际化。

（3）流程图组件库：使用twaver设计流程图组件，展示整个流程的流转过程，可进行流程模板的设计，各个环节属性的配置以及播放当前流转的情况并支持国际化。

（4）业务组件库：为部门设计公共业务组件库，如可配置表格，类型下拉树，并提供支持表单定制所需要的公共事件并支持国际化。

（5）项目模板：提供开发新版流程表单的项目模板，主要采用vue-cli2框架，支持多页面开发，支持单模块/多模块/全量编译，单模块/多模块/全量打包。 流程中心：提供表单编译脚本，并集成表单定制工具和流程图工具，实现表单定制工程化。
